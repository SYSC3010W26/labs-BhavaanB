#SYSC 3010-Labs

## Labs Overview
This course has 5 main labs we learn in this course.

# Lab 1
The lab teaches use the basics of settng up the lab content and experiments on a mini application to show how python connects with
the raspbery pi

# Lab 2 
The lab teaches us about databases and how to use sqlite in our applications and interact with python and raspberry pi

# Lab 3

The lab teaches us about the use of firebase and flask and how we can use a mapped out grid system to use as input which raspberry pi will execute the 
same commands based on that.

# Lab 4

The lab teaches us about how to use breadboards and familiarizes us with fritz software

# Lab 5
The lab teaches us the use of pi cams and how we interact with them in application using python.
